# Accessibility Inventory

Target: WCAG 2.2 Level AA  
Updated: 2026-06-20

## Routes

- Home: studio overview, hero media, venture cards, primary calls to action, and floating access launcher.
- Work: portfolio cards, external destinations, logo overlays, heading order, and floating access launcher.
- Adventures: brand lockup, book list, storefront destination, and floating access launcher.
- About: portrait, signature, profile content, and floating access launcher.
- Contact: inquiry form, validation, success, failure behavior, and floating access launcher.
- Privacy and Terms: reading order, contrast, reflow, and floating access launcher.
- Accessibility: support path, current accessibility information, and floating access launcher.
- 404: recovery links, unavailable-page handling, and floating access launcher.

## Shared components

- SiteHeader: named navigation, current page state, and mobile menu behavior.
- SkipLink: bypass repeated navigation.
- SiteFooterAccessible: navigation groups, support information, and permanent text-based preferences trigger.
- AccessibilityPreferences: native dialog, saved display settings, and focus return.
- AccessibilityPreferencesDock: fixed lower-left launcher, minimize action, recoverable restore tab, local-device launcher state, and dialog trigger.
- ContactForm: labels, autocomplete, validation, and status messages.
- Work cards: image alternatives, destination names, and new-tab disclosure.
