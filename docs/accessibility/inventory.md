# Accessibility Inventory

Target: WCAG 2.2 Level AA
Updated: 2026-06-20

## Routes

- Home: studio overview, hero media, venture cards, and primary calls to action.
- Work: portfolio cards, external destinations, logo overlays, and heading order.
- Adventures: brand lockup, book list, and storefront destination.
- About: portrait, signature, and profile content.
- Contact: inquiry form, validation, success, and failure behavior.
- Privacy and Terms: reading order, contrast, and reflow.
- Accessibility: support path and current accessibility information.
- 404: recovery links and unavailable-page handling.

## Shared components

- SiteHeader: named navigation, current page state, and mobile menu behavior.
- SkipLink: bypass repeated navigation.
- SiteFooter: navigation groups, support information, and preferences trigger.
- AccessibilityPreferences: native dialog and saved display settings.
- ContactForm: labels, autocomplete, validation, and status messages.
- Work cards: image alternatives, destination names, and new-tab disclosure.
