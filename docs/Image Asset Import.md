# Portfolio Image Asset Import

The portfolio pages on this branch reference optimized WebP assets in `public/images/`.

## Required files

```txt
public/images/
├─ home-hero-brandmark.webp
├─ about-portrait-context.webp
├─ work-xbeton-architecture.webp
├─ work-aff-dashboard.webp
├─ work-adventures-mockup.webp
├─ work-mercury-telephony.webp
├─ work-establishments-collage.webp
└─ og-social-background.webp
```

## Placement

| Asset | Used on |
| --- | --- |
| `home-hero-brandmark.webp` | Home hero |
| `about-portrait-context.webp` | About portrait feature |
| `work-xbeton-architecture.webp` | Home and Work: XBeton |
| `work-aff-dashboard.webp` | Home and Work: Advantage First Financial |
| `work-adventures-mockup.webp` | Home, Work, and Adventures pages |
| `work-mercury-telephony.webp` | Home and Work: Mercury Call Desk |
| `work-establishments-collage.webp` | Home and Work: 60+ Establishments |
| `og-social-background.webp` | Site-wide Open Graph and X/Twitter sharing metadata |

## Important

The page code is configured for these paths. Add the asset files before merging this branch so the Vercel preview and production deployment can display the images.
