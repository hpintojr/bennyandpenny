# Accessibility Statement

Canonical page: https://www.bennyandpenny.com/accessibility

Benny & Penny is working toward WCAG 2.2 Level AA.

## Preferences

Use the floating lower-left accessibility icon to adjust text size, contrast, motion, and font readability. The icon may be minimized, but a visible Accessibility restore tab remains available. The footer Accessibility preferences button opens the same native dialog.

## Support

Email hello@bennyandpenny.com to request an alternative format or report an accessibility barrier.
